24.1 Introduction

  • 私たちは対象についてのふわっとした知識を持っている
  • その知識を形式的で定量的なモデルへと変換する
  • そのモデルを使って以下が可能になる
    • 我々の知識を使った予測
    • データに潜む新たな知識の発見

別のアプローチ（機械学習）

  • 予測精度をひたすら高める
  • データが複雑すぎる場合はしゃーない？
  • ブラックボックスになりがち
    • うまくいく理由も条件も分からない
    • 将来大きな変化があっても機能するだろうか？

24.2 なぜ低品質なダイヤモンドの方が高価なのか？

ダイヤモンドの品質は次の4Cで決まる(はず)

  • carat: ダイヤモンドの重さ
  • cut: カット（形状、仕上がり）
  • color: 色。一般的には色が無い方が偉い。Dが最も無色でZに向かって色有り
  • clarity: 透明度。I1, SI2, SI1, VS2, VS1, VVS2, VVS1, IFの順に良い

cutが悪い方が高価？

ggplot(diamonds, aes(cut, price)) + geom_boxplot()

colorが悪い方が高価？

colorはDが一番良くて、Jが一番悪い

ggplot(diamonds, aes(color, price)) + geom_boxplot()

clarityが悪い方が高価？

ggplot(diamonds, aes(clarity, price)) + geom_boxplot()

24.2.1 価格とカラット

交絡因子カラット

ggplot(diamonds, aes(carat, price)) + geom_hex(bins = 50)

方針

  • モデルを使って価格を決める最大の因子 carat の影響を除去する
  • carat の影響を除去したときに他の因子が価格にどう影響するかを調べる

準備

diamonds2 <- diamonds %>% 
  filter(carat <= 2.5) %>%  #低カラットに絞り込む(全体の99.7%)
  mutate(lprice = log2(price), lcarat = log2(carat)) #対数変換

可視化

モデル分析
