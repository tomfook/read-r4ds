モデル予測の誤差

## RMSE
measure_distance <- function(mod, data) {
  diff <- data$y - model1(mod, data)
  sqrt(mean(diff ^2))
}
measure_distance(c(7, 1.5), sim1) 
## [1] 2.665212

\[ \mathrm{RMSE} = \sqrt{\frac{1}{N}\sum_{i=1}^N \left(y_i - (a_0 + a_1 x)\right)^2} \]

線形モデルを使った方法

sim1_mod <- lm(y ~ x, data = sim1)
coef(sim1_mod)
## (Intercept)           x 
##    4.220822    2.051533

線形モデルは解析的に解けるので最適解を直接計算できる

\[\begin{eqnarray} \hat a_0 &=& \bar{y} - \hat a_1\bar{x}\\ \hat a_1 &=& \frac{\sum_{i=1}^N(x_i-\bar x)(y_i-\bar y)}{\sum_{i=1}^N(x_i-\bar x)^2} \end{eqnarray}\]
